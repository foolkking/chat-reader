# Acceptance Criteria

## MVP Acceptance

- [ ] 可以运行 web + api + db。
- [ ] 可以上传 ChatGPT Exporter JSON。
- [ ] 可以上传 Markdown 并与 JSON 配对。
- [ ] 可以上传官方 conversations.json 并导入主线。
- [ ] 可以保存 source_artifacts。
- [ ] 可以保存 source_message_refs。
- [ ] 可以清洗思考摘要。
- [ ] 可以生成 Canonical Conversation Format。
- [ ] 可以生成 RenderBlock、Heading、SearchDocument。
- [ ] 可以查看会话列表。
- [ ] 可以阅读 Prompt / Response。
- [ ] 可以搜索并跳转。
- [ ] 可以创建 Project 并移动会话。
- [ ] 可以保存继续阅读位置。
- [ ] 大对话基础虚拟滚动可用。

## Release Acceptance

- [ ] 所有 Stage 验收通过。
- [ ] 关键路径 E2E 通过。
- [ ] 大文件导入不阻塞主线程。
- [ ] XSS 测试通过。
- [ ] 分享 token 安全。
- [ ] 导出文件可重新导入。
- [ ] 文档完整。
