# 产品愿景：AI 对话档案馆

## Overview

ChatGPT 导出对话阅读器的长期愿景，是把用户散落在 ChatGPT 导出文件中的对话历史，整理成可长期阅读、检索、编辑、合并、分享和归档的 AI Conversation Library。

它不是聊天机器人，也不是简单 JSON Viewer，而是围绕“对话知识资产”建立的一套信息管理系统。

## 核心价值

1. **长期保存**：摆脱原始导出格式和第三方工具格式限制。
2. **高质量阅读**：以 ChatGPT 风格展示 Prompt / Response。
3. **知识整理**：通过 Project、Tag、Pin、Search 管理大量对话。
4. **可编辑演化**：在不破坏原始记录的基础上编辑、插入笔记、合并、拆分。
5. **可迁移分享**：导出 Canonical JSON / Markdown / HTML。

## 用户类型

- 长期使用 ChatGPT 的个人用户。
- 需要整理学习、科研、工作对话的人。
- 需要把对话变成知识库的人。
- 需要从官方导出中恢复、查看、搜索历史的人。

## 不做什么

- 不做新的聊天机器人。
- 不复刻 ChatGPT 生成能力。
- 不执行原始导出内容中的脚本。
- 不把官方 conversations.json 直接作为运行时数据库结构。
